## Overview

---

## Essay Summary

The essay argues that computer networks are not just the tangled cables in a server room — they are the invisible circulatory system of modern daily life.

**Two main examples explored:**

### 1. Social Networks & Communication
Every morning check of a smartphone involves a chain of invisible network activity: data is packetized, transmitted over Wi-Fi or 5G, routed through ISP servers, and returned to the screen — all in milliseconds. When this pipeline is unstable, the symptoms are familiar: lag, failed sends, dropped connections. The fix is equally familiar: reset the signal or reconnect.

### 2. Navigation & Real-Time Data
Using Google Maps requires the app to communicate with satellites and ground-based networks simultaneously to determine location and pull live traffic conditions. A dead zone or network outage breaks this loop entirely — the app cannot reroute. This illustrates just how deeply dependent daily movement has become on network stability.

---

## Key Reflection

> The essay concludes that we depend on network reliability in the same way we depend on electricity or running water — we only truly notice it when it fails.
